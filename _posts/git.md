## Git Commands

Delete a file or folder in the repository:  
`git rm -r --cached filename`  
`git rm -r --cached foldername`

Remove all files from repository:  
`git rm -r *`  
`git push --set-upstream origin main`

Create a new branch:  
`git branch my-new-branch`
